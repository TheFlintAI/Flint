## Flint 是什么？

Flint 是一款多智能体应用，让你以最简单的方式，解决最复杂的问题。

<p align="center">
  <img src="docs/assets/demo.gif" alt="Flint Demo" />
</p>

## 特性

- **多智能体**：智能体自主规划、分工、协作，合力解决复杂问题。
- **模块化**：强大的插件系统，插件作者可自定义 UI、介入多智能体工作流、为 AI 提供更强大的工具以应对更广泛的场景。
- **经济**：全链路上下文优化，显著提升 Token 缓存命中率，节省成本。
- **简单**：保留真正有用的功能，化繁为简，开箱即用。
- **开放**：兼容所有主流模型供应商，自由选择你喜欢的模型。
- **轻量**：基于 Tauri 构建，内存占用低，性能卓越。

## 插件

Flint 从设计之初就充分考虑了扩展性。例如，内置的股票插件可以自主绘制行情面板，为 Agent 提供金融分析工具，将实时市场洞察无缝融入你的工作流。

## 快速开始

```bash
git clone https://github.com/TheFlintAI/Flint.git
cd Flint
bun install
bun run dev
```

需要 [Bun](https://bun.sh) 和 [Rust](https://rustup.rs)。
